# Launch Countdown Timer

A sleek countdown timer component perfect for product launches, events, or special occasions.

![Countdown Timer Screenshot](./screenshot.png)

**🔗 Live Demo:** [View Live Demo](https://kevinwangombe.github.io/launch-countdown-timer)

## ✨ Features

- Real-time countdown display
- Days, hours, minutes, seconds
- Target date configuration
- Responsive design
- Clean, modern aesthetics
- Smooth animations

## 🛠️ Tech Stack

- HTML5
- CSS3 (Animations, Flexbox)
- Vanilla JavaScript
- Date/Time API

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/kevinwangombe/launch-countdown-timer

# Open index.html in browser
# Or use a local server:
npx serve
```
